## Introduction

Website Contact Gatherer is a versatile scraper built to extract key communication details from websites. It solves the challenge of manually gathering contact information scattered across pages, making it ideal for lead generation, competitor research, and outreach workflows.

### Why This Tool Matters

- Automates discovery of emails, phone numbers, and profile links hidden across web pages.
- Helps outreach teams gather contact points at scale.
- Reduces manual workload by structuring all results into export-ready formats.
- Ideal for anyone needing accurate business or creator contact information.
- Works on virtually any public website.

## What Data This Scraper Extracts

| Field Name | Field Description |
|------------|------------------|
| email | Extracted email addresses found on the website. |
| phone | Phone numbers detected anywhere on the site. |
| facebook_profile | Direct link to any Facebook page or profile found. |
| instagram_profile | Direct link to identified Instagram profiles. |
| linkedin_profile | Company or personal LinkedIn URLs found. |
| twitter_x_profile | Links leading to Twitter/X accounts. |
